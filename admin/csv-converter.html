CSV converter

Convert Google Sheets CSV into ESAT question objects.

Paste CSV exported from Google Sheets, validate the required columns, then copy the generated JavaScript. This page is fully static, uses no external libraries, and works locally or on GitHub Pages.

1. Paste CSV

Required headers are case-sensitive. Quoted commas and multi-line cells are supported. The section column accepts maths1, maths2, physics, or common labels such as Maths 1.

Paste CSV to begin.

Column guide

  • Required: internalId, section, topicCode, topicName, questionText, optionA through optionE, correctAnswer, workedSolution.
  • Optional source fields: sourceExam, sourceYear, sourcePaper, sourceQuestionNumber.
  • Optional teaching fields: quickestMethod, commonTrap, difficulty, tags.
  • Optional image fields: imagePath, imageAlt.
  • Correct answer: use A, B, C, D, E, or 1 to 5.
  • Tags: separate multiple tags with commas, semicolons, or pipes.
ESAT fields id, subject, topic, difficulty, question, options, answerIndex, explanation, tags
Admin metadata topicCode, sourceExam, sourceYear, sourcePaper, sourceQuestionNumber, quickestMethod, commonTrap, imagePath, imageAlt
