## Worked solution

Add 6 to both sides to get x/2 < 14, then multiply by 2 to get x < 28.

## Quickest method

Move the constant first: x/2 < 14, so x < 28.

## Common traps

Multiplying only one side by 2 or treating the displayed fraction as 2x. The inequality sign does not reverse because the multiplication is by positive 2.

## Why the wrong options are wrong

- A. x < 4: Too small; this would require 2x < 8.

- B. x > 4: Wrong inequality direction and value.

- C. x < 20: Too small; this would require 2x < 40.

- D. x > 20: Wrong inequality direction and value.

- E. x < 22: Too small; this would require 2x < 44.
