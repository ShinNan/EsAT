## Worked solution

Add 68 to both sides to get 2x < 56, then divide by 2 to get x < 28.

## Quickest method

Move the constant term first: 2x < 56, so x < 28.

## Common traps

Forgetting that only multiplying or dividing by a negative reverses an inequality; here division is by positive 2 so the sign stays the same.

## Why the wrong options are wrong

- A. x < 4: Too small; this would require 2x < 8.

- B. x > 4: Wrong inequality direction and value.

- C. x < 20: Too small; this would require 2x < 40.
