CSV converter

Convert Google Sheets CSV into a full ESAT question bank file.

Paste CSV exported from Google Sheets, validate the schema, then copy a complete replacement assets/questionBank.js file. This page is fully static, uses no external libraries, and works locally or on GitHub Pages.

1. Paste CSV

Required headers are case-sensitive. Quoted commas and multi-line cells are supported. Use either section or subject for the runtime subject; accepted values are maths1, maths2, physics, or common labels such as Maths 1.

Paste CSV to begin.

Column guide

  • Required: internalId, section or subject, topicCode, topicName, questionText, optionA through optionE, correctAnswer, workedSolution.
  • Optional teaching fields: difficulty, quickestMethod, commonTrap, tags, status.
  • Optional source fields: sourceExam, sourceYear, sourcePaper, sourceSection, sourceQuestionNumber, originalReference.
  • Optional image fields: imagePath, imageAlt.
  • Correct answer: use A, B, C, D, or E.
  • Difficulty: use 1 to 5, or easy, medium, hard.
  • Tags: separate multiple tags with commas, semicolons, or pipes.
