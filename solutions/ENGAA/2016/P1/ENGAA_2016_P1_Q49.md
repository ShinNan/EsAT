## Worked solution

The cursor has height \(10\) and moves in the negative \(x\)-direction, so only \(x\le0\) is relevant.

For the curve

\[
y=4x^{3}-12x^{2}-36x-15,
\]

differentiate:

\[
y'=12x^{2}-24x-36=12(x-3)(x+1).
\]

The stationary point in the relevant region \(x\le0\) is \(x=-1\).

At this point,

\[
y=4(-1)^{3}-12(-1)^{2}-36(-1)-15=5.
\]

This is the highest point of the curve for \(x\le0\), so the smallest vertical distance from the cursor height \(10\) to the graph is

\[
10-5=5.
\]

## Quickest method

Maximise the graph value for \(x\le0\), then subtract it from \(10\).

## Common trap

Looking for the nearest point on the whole graph rather than restricting to the cursor moving left from \(x=0\).
